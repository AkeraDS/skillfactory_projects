Данные в датасете:
1. flight_id - идентификатор рейса
2. actual_departure - фактическая дата отбытия рейса
3. departure_city - город отбытия
4. arrival_city - город прибытия
5. model - модель самолета
6. purchased_tickets - количество купленных билетов
7. business - количество купленных мест бизнес-класса
8. economy - количество купленных мест эконом-класса
9. amount_tickets - общая стоимость всех проданных билетов
10. minute_in_way - количество минут в пути
11. fuel_consumption - расходы на топливо
12. profit - прибыль от рейса, рассчитанная по формуле стоимость_билетов - расходы_на_топливо
